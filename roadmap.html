Cyber Security Analyst

Cyber Security is an increasing popular industry for students in the IT industry with an increase in job growth in future.
It focuses on planning and implementing security measures to protect computer networks and systems.

  • Knowledge on attacks, malware, firewall, encryption.
  • Reporting breaches and vulneralbilites.
  • Educating other employees in the company on risks.
  • Foresic investigating
  • Identifying potential security risks.
  • Basic understanding of most coding languages.
  • Working within a team.
  • Learning and delivering best security practices.
  • Knowledge of concepts and architecture in Networking.
  • Understanding of Cloud.

Web Developer - Full Stack

To become a full stack Web Developer, you need to learn front-end and back-end technical skills. This involves:

  • HTML, CSS, Javascript, PHP, and JSON as just some of the main coding languages.
  • CSS frameworks like Bootstrap.
  • SQL databases or non-SQL databases for example MongoDB.
  • Designing Web pages taking into account User-Centered design.
  • Back-end skills such as Python, Java, C, C++.
